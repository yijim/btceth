Lesson 4: CEX vs DEX

Time: 20 min · Last updated: 2025-11-14

1. CEX & DEX in one sentence

In crypto, there are two main ways to trade: CEX (Centralized Exchange，中心化交易所) and DEX (Decentralized Exchange，去中心化交易所).

A simple mental model:
CEX ≈ 你把钱存进交易所账户里，让交易所帮你撮合买卖；
DEX ≈ 你用自己的链上钱包，直接通过智能合约交换代币。

2. CEX vs DEX at a glance

先用一张表把两者的感觉拉开：

CEX（中心化交易所）

  • 典型例子：Binance、OKX、Bybit 等
  • 你先把法币/数字货币充进交易所账户
  • 交易所维护一套「内部账本」，帮你撮合挂单
  • 有账号密码、邮箱、手机号、KYC、客服
  • 下单体验像炒股票：限价、市价、止盈、止损、合约

DEX（去中心化交易所）

  • 典型例子：Uniswap（以太坊）、Jupiter / Raydium（Solana）
  • 资产始终在你的链上钱包里（Metamask、Phantom 等）
  • 通过 智能合约 直接和流动性池做交换
  • 不需要账号、密码、邮箱，只有「钱包授权」
  • 上币速度快，长尾资产多，尤其是新发行的 meme / 小币

小结：CEX 更像“带客服的证券账户”，DEX 更像“你自己掌握钥匙的钱包+换币工具”。

3. How CEX works（CEX 是怎么运作的？）

在 CEX 里，你看到的「余额」，本质上是交易所在自己的数据库里，给你记的一行账。 真正的链上资产，由交易所统一管理在它自己的冷/热钱包里。

  • 你充值：链上转账 → 交易所检测到账 → 把内部账户余额 +1
  • 你下单：只是改数据库里的「挂单表」「成交记录」
  • 你提现：交易所从它的钱包往你的地址打币

优点：体验好、撮合快、功能多（合约、杠杆、止损等）。
风险：你在信任交易所，它不跑路、不爆雷、不乱冻结。

4. How DEX works（DEX 是怎么运作的？）

在 DEX 里，你的钱不进「平台账户」，而是一直在你的钱包地址里。 交易靠的是链上的智能合约 + 流动性池。

  • 有人把 Token A 和 Token B 存进池子（提供流动性）
  • 你想用 A 换 B：调用合约，合约按公式（如 x·y=k）帮你兑换
  • 所有记录直接写在链上，可查可追踪

优点：非托管（non-custodial），你自己掌握私钥；任何人都可以接入，不需要审批。
风险：合约有可能有漏洞、项目方可以改参数、假币合约很多。

5. Pros & cons（优缺点对比）

CEX 优点

  • 支持法币充值/提现，适合第一笔入金
  • 下单界面友好，新手上手成本低
  • 有客服、工单、风控、风控提示
  • 衍生品丰富：合约、杠杆、期权等

CEX 缺点

  • 需要 KYC，存在隐私和合规风险
  • 平台有权冻结账号或限制提现
  • 极端情况下有「跑路 / 被黑客攻击」风险

DEX 优点

  • 自己掌握私钥，资产不托管给平台
  • 全球可用，不需要实名注册
  • 上币速度快，能第一时间接触到新项目
  • 适合链上原生玩法（meme、NFT、DeFi 等）

DEX 缺点

  • 需要懂基础链上操作，新手容易点错
  • 合约 / 假币 / 钓鱼网站风险高
  • 极端行情时滑点大、gas 费可能飙升

6. When should you use which?（什么时候用哪个？）

给一个简单、可执行的决策模板，你可以按自己风格微调：

  • 第一笔入金（法币 → USDT / BTC / ETH）：多数情况下用 CEX。
  • 玩 meme、小币、刚上的新项目：必须用 DEX + 链上钱包。
  • 高频合约、短线交易：一般还是 CEX 体验更好、工具更多。
  • 中长期持仓（BTC / ETH / SOL 等主流币）：考虑冷钱包或多地址分散保管。

很多人实际的路径是：
法币 → CEX 买主流币 → 提一部分到钱包 → 用 DEX 试水小仓位

⚠️ Safety checklist（安全清单）

在 CEX：

  • 不要把所有资产都放在一家交易所
  • 必须开启 2FA（Google Auth / 手机验证）
  • 邮箱、手机卡、登录密码分开管理

在 DEX：

  • 助记词只手抄，永远不要截屏 / 发送 / 网盘保存
  • 每次连接钱包前确认网址是否正确，提防钓鱼站
  • 新合约先小额试单，避免被假币、蜜罐套路

7. Quick recap（本节小结）

  • CEX：中心化交易所，体验好，但需要信任平台。
  • DEX：去中心化交易所，你掌握私钥，但要对自己的操作负责。
  • 实际使用里，两者并不是二选一，而是搭配使用。
