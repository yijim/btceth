Lesson 2: Public Chains & Layer 2

Time: 20 min · Last updated: 2025-11-14

1. What is a Public Chain?

A public chain（公链） is an open blockchain network that anyone can use, build on, or send transactions on.

你可以把公链简单理解为：一个全球共享的计算机。

  • Bitcoin = 最安全的价值存储链
  • Ethereum = 智能合约的标准平台
  • Solana = 高性能链，速度快、费用极低

公链负责处理所有链上活动：转账、部署合约、Mint NFT、Swap 等等。 每条链的交易规则、速度、费用都不同。

2. Why do we need different public chains?

因为每条链有不同的设计哲学和侧重点：

Bitcoin · 安全第一

  • 去中心化程度最高
  • 设计简单且极其稳健
  • 主要用途：价值存储（Digital Gold）

Ethereum · 灵活与生态

  • 智能合约发源地
  • DeFi / NFT / DAO 的主阵地
  • 但手续费（Gas）较贵，需要扩容

Solana · 高性能链

  • 速度非常快（快速出块）
  • 费用极低，适合高频、小额交易
  • Meme 文化火爆，生态发展快

小结：不同链像不同城市，各有定位、节奏和文化。

3. So what is Layer 2 (L2)?

Layer 2 是专门为 Ethereum 扩容的网络。 它在不牺牲安全性的前提下，让交易变得更快、更便宜。

简单说：ETH 太拥挤 → L2 来帮忙减压

  • 把大量交易放到链下处理
  • 只把最终结果提交回以太坊主网（L1）
  • 因此：手续费大幅降低，速度大幅提升

典型 L2：

  • Arbitrum – 目前最大 L2
  • Optimism – 和 Coinbase 合作 Base
  • Base – Meme 生态很强（包括 BRETT）
  • zkSync – zk 技术路线

4. Public Chains vs Layer 2

一个非常简单、直观的对比：

Public Chain（L1）

  • 主链，安全性最高
  • 交易写在主链上，成本较高
  • 速度较慢（尤其以太坊）
  • 适合价值存储或大型协议

Layer 2（L2）

  • 构建在 L1 之上
  • 交易便宜很多
  • 速度更快
  • 适合日常交易、Meme、交互

概念理解：L1 是高速公路，L2 是匝道。

5. Why does this matter for you?

因为不同的链，直接影响你的：

  • 转账费用（Gas）
  • 转账速度
  • 你能玩到的项目和生态
  • 钱包地址是否兼容（EVM / non-EVM）

举例：

  • 你玩 Solana meme → 一般用 Phantom + Solana 主网
  • 你玩 ETH NFT → 一般在 Ethereum 或 Base
  • 你跨链转资产 → 可能需要桥（Bridge）

⚠️ Watch out for network mistakes

新手最常见的错误：发错网络。

  • 把 ETH 发送到 Solana 地址（直接消失）
  • 把 USDT-ERC20 当成 USDT-TRC20 处理
  • 忘记切链就授权 / Swap

规则：永远确认你当前使用的是哪条链。

6. Quick recap

  • Public Chain = 链的底层（BTC / ETH / SOL）
  • Layer 2 = 为以太坊扩容的“更便宜更快的链”
  • 不同链有不同生态：NFT、DeFi、Meme、合约、跨链
  • 你的钱包和 Gas 会随着链不同而变化
